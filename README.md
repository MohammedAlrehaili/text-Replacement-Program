# Text Replacement Program

A lightweight, web-based utility designed to search for specific words within a text block and replace them dynamically. This project is part of my learning journey in the JavaScript 103 course provided by Satr.codes.

## 🚀 Features

  * **Dynamic Search:** Finds all occurrences of a word using global flags.
  * **Instant Replacement:** Updates the UI immediately without reloading the page.
  * **Arabic Support:** Fully compatible with RTL (Right-to-Left) text and Arabic characters.
  * **Responsive Design:** Clean and simple interface suitable for various screen sizes.

## 🛠️ Tech Stack

  * **HTML5:** Structure and content.
  * **CSS3:** Custom styling and layout.
  * **JavaScript (ES6):** Logic, DOM selection, and Regular Expressions.

## 📖 How to Use

1.  Enter the word you want to find in the **"Search"** field.
2.  Enter the new word in the **"Replace"** field.
3.  Click the **"Replace" (استبدل)** button.
4.  The text area below will automatically update with the new content.
